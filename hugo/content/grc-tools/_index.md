Open-source GRC (Governance, Risk, and Compliance) policy templates, tools, and guidance. Built from real-world deployment experience and aligned with NIST 800-53, SOC 2, and ISO 27001.

## Start here

If you are building a security program from scratch, start with the [Getting Started guide](/grc-tools/guides/getting-started/).

## What's here

### [Guides](/grc-tools/guides/)

Tool evaluation guides and implementation roadmaps. Vendor-neutral comparisons of security tools across categories like log management, SIEM, endpoint protection, and more. Plus a framework mapping showing every policy's alignment to NIST, SOC 2, and ISO 27001.

### [Policy Templates](/grc-tools/policy-templates/)

32 production-ready policy templates in ISP format. Every policy comes with a fill-in-the-blanks template (what is required), a companion procedure document (how to implement), and a README with plain-English explanation and gotchas.

### [Resources](/grc-tools/resources/)

Operational templates: vendor security questionnaire, risk register template, annual policy review checklist, and evidence collection guide for audits.
